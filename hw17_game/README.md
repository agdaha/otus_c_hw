# Игра 2048

С использованием библиотеки Allegro 5.

- `liballegro`
- `liballegro_font`
- `liballegro_ttf`
- `liballegro_primitives`
- `liballegro_image`

Графическая версия игры 2048 с возможностью сохранения таблицы лидеров.
Игрок управляет плитками на поле 4×4, перемещая их в четырёх направлениях. При столкновении
одинаковые плитки объединяются, увеличивая счёт вдвое. Цель — собрать  2048.

## Сборка и запуск

```bash
# Сборка
make

# Запуск
./game2048

# Очистка артефактов сборки
make clean
```

## Сохранение рекордов

Результаты сохраняются в бинарный файл `records.dat` в текущей директории.
Хранится до 10 лучших результатов, отсортированных по убыванию счёта.
