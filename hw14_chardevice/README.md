# Домашнее задание

Драйвер символьного устройства

## Цель:

Научиться писать простые драйверы.

## Описание/Пошаговая инструкция выполнения домашнего задания:

    Создать модуль ядра;
    Добавить чтение и запись;
    Сделать кольцевой буфер;
    Добавить настройку через sysfs;
    Протестировать.

## Требования к ДЗ:

    Корректная работа операций чтения/записи;
    Реализация кольцевого буфера;
    Корректная работа с sysfs;
    Качественное оформление кода и документации.

---

## Инструкция по сборке и запуску

### 1. Подготовка окружения

Убедитесь, что установлены заголовочные файлы ядра.

### 2. Сборка модуля

В директории с проектом выполните команду:

```bash
make
```

### 3. Загрузка в ядро

```bash
sudo insmod ring_buffer.ko
```

Проверьте успешность загрузки через `dmesg`:

```bash
dmesg | tail
```

---

## Тестирование

### Проверка записи и чтения

Откройте терминал и запишите данные:

```bash
echo "Hello, ring buffer" > /dev/ring_buffer_dev
```

Прочитайте данные:

```bash
cat /dev/ring_buffer_dev
```

**Права доступа**: Если cat или echo выдают "Permission denied", дайте права файлу: `sudo chmod 666 /dev/ring_buffer_dev`.

### Работа с sysfs

Посмотреть текущий размер буфера:

```bash
cat /sys/kernel/ring_buffer_kobj/buf_size
```

Изменить размер буфера до 2048 байт (данные в буфере будут сброшены):

```bash
echo 2048 | sudo tee /sys/kernel/ring_buffer_kobj/buf_size
```

## Удаление модуля

```bash
sudo rmmod ring_buffer
make clean
```
