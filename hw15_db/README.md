## Домашнее задание

    Статистика БД

#### Цель:

    В процессе выполнения ДЗ вы получите опыт взаимодействия с базами данных.

#### Описание/Пошаговая инструкция выполнения домашнего задания:

    Написать программу, вычисляющую статистические параметры в заданной таблице БД по заданной колонке:

* среднее
* максимум
* минимум
* дисперсия
* сумма

#### Требования:

1. Создано консольное приложение, принимающее аргументами командной
   строки название базы данных, название таблицы и название колонки.
2. Приложение корректно вычисляет требуемые значения по заданной колонке.
3. Приложение корректно обрабатывает нечисловой тип данных в заданной колонке.
4. Приложение корректно обрабатывает ошибки доступа к БД.
5. **Бонусные баллы** за поддержку различных движков БД.
6. Код компилируется без предупреждений с ключами компилятора `-Wall -Wextra -Wpedantic -std=c11`.
7. Далее успешность определяется ревью кода.

# Реализация

## Возможности

- **Поддержка двух СУБД**: SQLite и PostgreSQL
- **Расчёт статистики**: среднее, максимум, минимум, сумма, дисперсия (выборочная)
- **Валидация входных данных**: проверка имён таблиц и колонок на допустимые символы
- **Обработка ошибок**:
  - Нечисловой тип данных в колонке
  - Ошибки подключения к БД
  - Пустая таблица или отсутствие данных

## Сборка

```bash
make
```

Требуемые библиотеки:

- `libsqlite3` — для работы с SQLite
- `libpq` — для работы с PostgreSQL

Очистка сборки:

```bash
make clean
```

## Запуск

```bash
./stat_db_table <type> <db_source> <table_name> <column_name>
```

Параметры:

- `type` — тип БД: `sqlite` или `postgres`
- `db_source` — путь к файлу БД (SQLite) или строка подключения (PostgreSQL)
- `table_name` — имя таблицы
- `column_name` — имя колонки для расчёта статистики

### Примеры

**SQLite:**

```bash
./stat_db_table sqlite ./mydb.sqlite users age
```

**PostgreSQL:**

```bash
./stat_db_table postgres "host=localhost dbname=mydb user=postgres" users age
```

## Пример вывода

```
--- Статистика по колонке [age] ---
Среднее:   35.5000
Максимум:  65.0000
Минимум:   18.0000
Сумма:     710.0000
Дисперсия (выборка): 225.0000
```
